# 猜数字游戏 - Python初学者笔记

## 1. 导入模块
```python
import random
```
- `random` 是 Python 的标准库，用于生成随机数
- `randint(a, b)` 函数可以生成 a 到 b 之间的随机整数（包含 a 和 b）

## 2. 函数定义
```python
def guess_number_game():
```
- `def` 关键字用于定义函数
- 函数名遵循 snake_case 命名规范（小写字母，下划线分隔）
- 函数是组织代码的基本单元，可重复使用

## 3. 变量与数据类型
```python
min_num = 1              # 整数 (int)
max_num = 100            # 整数 (int)
max_attempts = 7         # 整数 (int)
success = False          # 布尔值 (bool)
guess_history = []       # 列表 (list)
```
- Python 是动态类型语言，不需要声明变量类型
- 列表用 `[]` 表示，可以存储多个元素

## 4. 循环结构

### while 循环
```python
while True:  # 无限循环
```
- `while` 循环在条件为 `True` 时持续执行
- `True` 是布尔值，表示永远为真
- 需要 `break` 语句来退出循环

### 嵌套循环
```python
while attempts < max_attempts:  # 外层循环：控制猜测次数
    while True:  # 内层循环：验证输入有效性
```
- 循环可以嵌套使用
- 内层循环负责输入验证，外层循环控制游戏流程

## 5. 条件判断

### if-elif-else 结构
```python
if guess < target:
    print("猜小了")
elif guess > target:
    print("猜大了")
else:
    print("答对了")
```
- `if`：条件成立时执行
- `elif`：额外的条件判断
- `else`：所有条件都不成立时执行

### 比较运算符
- `<`：小于
- `>`：大于
- `==`：等于
- `<=`：小于等于
- `>=`：大于等于

### 逻辑运算符
```python
if min_num <= guess <= max_num:  # 连续比较
```
- Python 支持链式比较
- `and`：与（同时满足）
- `or`：或（满足其一）
- `not`：非（取反）

## 6. 异常处理
```python
try:
    guess = int(input("输入数字："))
except ValueError:
    print("请输入整数")
```
- `try-except` 用于捕获和处理异常
- `ValueError`：当无法转换为整数时触发
- 防止程序因错误输入而崩溃

## 7. 字符串格式化

### f-string (Python 3.6+)
```python
print(f"输入一个{min_num}~{max_num}之间的数字")
```
- 在字符串前加 `f` 或 `F`
- 用 `{}` 包裹变量名，会自动替换为变量值
- 最常用、最简洁的格式化方式

### 字符串连接
```python
print("→".join(guess_history))
```
- `join()` 方法用指定字符串连接列表元素
- 示例：`["10↑", "20↓"]` → `"10↑→20↓"`

## 8. 列表操作
```python
guess_history = []           # 创建空列表
guess_history.append(f"{guess}↑")  # 添加元素到末尾
"→".join(guess_history)      # 连接列表元素
```
- `append()`：在列表末尾添加元素
- 列表是可变的，可以动态添加元素

## 9. 输入输出

### 输出
```python
print("游戏")  # 打印文本
print(f"一共尝试了{attempts}次")  # 打印格式化字符串
```

### 输入
```python
guess = int(input("输入你猜的数字："))
choice = input("继续吗？")
```
- `input()` 获取用户输入，返回字符串
- `int()` 将字符串转换为整数

## 10. 主程序入口
```python
if __name__ == "__main__":
    guess_number_game()
```
- `__name__` 是 Python 的特殊变量
- 当文件直接运行时，`__name__` 等于 `"__main__"`
- 当文件被导入时，`__name__` 等于文件名
- 这个结构让文件既可以作为模块导入，也可以直接运行

## 11. 游戏逻辑总结
1. 生成 1-100 的随机目标数
2. 玩家有 7 次猜测机会
3. 输入验证：确保输入的是有效范围内的整数
4. 提示猜大了还是猜小了
5. 记录猜测历史
6. 游戏结束后显示结果
7. 询问是否重新开始

## 12. 常用技巧
- `break`：退出当前循环
- `+=`：自增运算符（`attempts += 1` 等同于 `attempts = attempts + 1`）
- `not`：逻辑非（`not success` 表示未成功）

## 13. 代码规范建议
- 运算符前后加空格（`a = 1` 而不是 `a=1`）
- 逗号后加空格
- 适当使用空行分隔代码块
- 变量名要有意义（如 `max_attempts` 比 `ma` 好）
