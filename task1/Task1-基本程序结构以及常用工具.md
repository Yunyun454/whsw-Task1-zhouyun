## 练习任务

### 1. 猜数游戏

#### 随机数生成规则

程序启动后，根据用户指定的区间自动生成一个随机数。**精度由区间端点的格式决定**：

| 区间示例 | 生成规则 |
|---|---|
| `1 - 100`（整数） | 生成随机**整数** |
| `1.00 - 100.00`（两位小数） | 生成随机**两位浮点数** |
| `1.000 - 1000.000`（三位小数） | 生成随机**三位浮点数** |
| `1.000 - 100.0`（最小值三位、最大值一位） | **以最小值精度为准**，生成随机**三位浮点数** |

简单来说：看区间的最小值有几位小数，就生成几位小数的随机数；如果最小值没有小数点，就生成整数。

#### 游戏流程

1. 设定一个最大重试次数。
2. 玩家输入猜测值：
   - 若**猜大了** → 提示"太大了"，告知剩余重试次数，让玩家重新输入。
   - 若**猜小了** → 提示"太小了"，告知剩余重试次数，让玩家重新输入。
   - 若**猜对了** → 提示"恭喜你猜对了！"，并告知总共尝试了多少次。
3. 若重试次数用完仍未猜中 → 提示"游戏结束"，并公布正确答案。
4. 每轮结束后，玩家可以选择**重新开始**，此时随机数重新生成，重试次数重置。
5. **历史记录**：记录玩家本轮每一次的猜测值。猜对或游戏结束时，展示本轮所有猜测轨迹，例如：`你的猜测轨迹：50 ↑ → 80 ↓ → 65 ↑ → 72 ↓ → 67 √`（用 ↑ 表示上次猜小了、↓ 表示上次猜大了），让玩家直观看到自己是如何一步步逼近答案的。

可以适当添加一些额外的功能，例如：让玩家设定范围、添加倒计时等等。自行添加描述输出，如：“请输入你猜的数字（例如：100）：:”。

### 2. 知识点清单

在开始写代码之前，建议先掌握以下知识点：

| 类别 | 知识点 | 在游戏中的应用 |
|---|---|---|
| **顺序结构** | 变量与数据类型、`input()` 输入、`print()` 输出 | 程序的整体执行流程 |
| **分支结构** | `if / elif / else` 条件判断 | 判断猜大了还是猜小了 |
| **循环结构** | `while` 循环、`break` 退出 | 反复让玩家猜测直到猜中或用完次数 |
| **随机数** | `random.randint()`、`random.uniform()` | 生成目标数字 |
| **字符串处理** | `split()`、`len()`、`isdigit()` | 判断小数点位数、校验输入是否合法 |
| **列表** | `list.append()`、遍历列表 | 存储历史猜测记录 |
| **异常处理** | `try / except` | 防止玩家输入非数字导致程序崩溃 |
| **函数** | `def` 定义函数、参数与返回值 | 将游戏逻辑拆分为独立函数（如生成随机数、判断大小） |
