# 猜数游戏学习笔记

## 一、本次任务学到的新知识点

### 1. 变量与数据类型

```python
min_val = 1.00        # 浮点数（小数）
max_attempts = 10     # 整数
name = "猜数游戏"     # 字符串
```

| 类型 | 说明 | 示例 |
|------|------|------|
| int | 整数 | `1`, `100`, `-5` |
| float | 浮点数（小数） | `1.00`, `3.14` |
| str | 字符串 | `"hello"`, `'你好'` |

### 2. 输入与输出

```python
# 输入：从键盘获取用户输入，返回的是字符串
name = input("请输入你的名字：")

# 输出：在屏幕上显示内容
print("你好！")
print(f"你的名字是 {name}")  # f-string 格式化字符串
```

**注意**：`input()` 返回的永远是字符串，需要用 `int()` 或 `float()` 转换。

### 3. 条件判断（if / elif / else）

```python
if guess == target:
    print("猜对了！")
elif guess < target:
    print("猜小了")
else:
    print("猜大了")
```

**比较运算符**：
- `==` 等于
- `!=` 不等于
- `>` 大于
- `<` 小于
- `>=` 大于等于
- `<=` 小于等于

### 4. 循环

#### while 循环
```python
while attempts < max_attempts:
    # 只要条件为真，就一直执行
    guess = input("请输入：")
    attempts += 1
```

#### break 退出循环
```python
while True:
    if 条件满足:
        break  # 跳出循环
```

### 5. 列表（List）

```python
# 创建空列表
history = []

# 添加元素
history.append((guess, "↑"))

# 遍历列表
for guess, result in history:
    print(guess, result)
```

### 6. 字符串处理

```python
text = "1.00"

# 分割字符串
parts = text.split('.')    # ['1', '00']

# 判断是否包含某字符
if '.' in text:            # True

# 获取长度
len("hello")               # 5

# 去除末尾字符
"abc → ".rstrip(" → ")     # "abc"
```

### 7. 异常处理（try / except）

```python
try:
    num = int(input("请输入数字："))
except ValueError:
    print("输入的不是有效数字！")
```

**作用**：防止用户输入错误导致程序崩溃。

### 8. 函数（def）

```python
def get_decimal_places(text):
    """获取小数位数"""  # 文档字符串
    if '.' in text:
        return len(text.split('.')[1])
    return 0

# 调用函数
result = get_decimal_places("1.00")  # 返回 2
```

### 9. 随机数

```python
import random

# 生成随机整数
num = random.randint(1, 100)

# 生成随机浮点数
num = random.uniform(1.0, 100.0)

# 四舍五入保留小数位
num = round(3.14159, 2)  # 3.14
```

## 二、代码思路和关键逻辑

### 整体流程图

```
开始
  ↓
设置猜测范围（最小值、最大值）
  ↓
设置最大尝试次数
  ↓
生成随机数
  ↓
循环：让玩家猜测
  ├─ 猜对了 → 显示恭喜 → 结束
  ├─ 猜小了 → 提示"太小了" → 继续
  └─ 猜大了 → 提示"太大了" → 继续
  ↓
次数用完 → 显示答案
  ↓
显示猜测轨迹
  ↓
询问是否再来一局
  ↓
结束
```

### 关键逻辑说明

#### 1. 如何判断小数位数？

```python
def get_decimal_places(text):
    if '.' in text:
        decimal_part = text.split('.')[1]  # 取小数点后的部分
        return len(decimal_part)           # 返回长度
    return 0
```

例如：`"1.00".split('.')` 得到 `['1', '00']`，小数部分是 `'00'`，长度为 2。

#### 2. 如何生成指定精度的随机数？

```python
if decimal_places == 0:
    # 整数：用 randint
    return random.randint(min_val, max_val)
else:
    # 小数：用 uniform + round
    return round(random.uniform(min_val, max_val), decimal_places)
```

#### 3. 如何记录猜测历史？

使用列表存储元组 `(猜测值, 结果符号)`：

```python
history = []
history.append((50, "↑"))   # 猜小了
history.append((80, "↓"))   # 猜大了
history.append((65, "="))   # 猜对了
```

## 三、遇到的问题及解决方法

### 问题1：input() 返回的是字符串

**问题**：用户输入 `100`，但 `input()` 返回的是字符串 `"100"`，不能直接比较大小。

**解决**：用 `float()` 或 `int()` 转换：
```python
guess = float(input("请输入数字："))
```

### 问题2：用户输入非数字导致程序崩溃

**问题**：用户输入 `abc`，`float("abc")` 会报错。

**解决**：用 `try / except` 捕获异常：
```python
try:
    guess = float(input("请输入数字："))
except ValueError:
    print("请输入有效的数字！")
```

### 问题3：如何让玩家可以重复玩？

**解决**：用 `while` 循环包裹整个游戏逻辑，玩家选择是否继续：
```python
while True:
    play_game()  # 玩一局
    again = input("再来一局？")
    if again != "yes":
        break
```
